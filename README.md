## 🚀 Featured Projects

### 🛸 [Autonomous Mission Logging and Fleet Coordination Database for UAV Systems](https://github.com/WorldlySage03/Autonomous-Mission-Logging-and-Fleet-Coordination-Database-for-UAV-Systems)
> **PL/SQL · Oracle SQL · Relational Schema Design**

A full relational database system for managing an autonomous delivery drone fleet. Features a normalized schema covering UAVs, warehouses, missions, real-time telemetry, and maintenance logs. Implements PL/SQL triggers with Haversine spherical distance calculations for automated mission assignment.

---

### 🎮 [Tic Tac Toe Desktop Application](https://github.com/WorldlySage03/Tic-Tac-Toe-built-with-Java-in-NetBeans-25)
> **Java · Java Swing · NetBeans 25**

A fully functional desktop game with single-player (vs AI) and two-player local modes. Features persistent score tracking, win/draw detection, and a complete Swing GUI. Packaged as a runnable JAR.

---

### 🤖 [Line Follower Robot with Obstacle Detection](https://github.com/WorldlySage03/Line-Follower-Robot-with-Obstacle-Detection-Made-with-Arduino)
> **C++ · Arduino UNO R3 · IR & Ultrasonic Sensors**

An autonomous robot that follows a black line using IR sensors and detects/avoids obstacles via an ultrasonic sensor. Includes circuit diagrams, wiring schematics, and complete documentation.
